## Glossary of Terms

### **A**
- **AI (Artificial Intelligence)**: Computer systems that perform tasks requiring human-like intelligence
- **API (Application Programming Interface)**: Interface for software communication
- **Autonomous**: Self-governing, operating without human intervention

### **B**
- **Balancing Control**: Maintaining robot stability during motion
- **Bipedal**: Having two legs for locomotion
- **Bridge**: Connection between simulation and real systems

### **C**
- **CUDA**: NVIDIA's parallel computing platform
- **Computer Vision**: AI field focusing on image understanding
- **Cloud Robotics**: Robots connected to cloud computing resources

### **D**
- **Deep Learning**: Machine learning with neural networks
- **Domain Randomization**: Technique for sim-to-real transfer
- **Dynamics**: Physics of motion and forces

### **E**
- **Embodied AI**: AI systems with physical presence
- **End-Effector**: Robot tool or hand at end of arm
- **Environment**: Robot's operational surroundings

### **F**
- **Foundation Model**: Large pre-trained AI model
- **Footstep Planning**: Planning where robot places feet
- **Fusion**: Combining multiple sensor modalities

### **G**
- **GPU (Graphics Processing Unit)**: Parallel processing hardware
- **Ground Truth**: Correct reference data for training
- **Gym**: Reinforcement learning environment

### **H**
- **Humanoid**: Human-shaped robot
- **Human-Robot Interaction**: Study of human-robot communication
- **Hardware**: Physical robot components

### **I**
- **Isaac**: NVIDIA's robotics platform
- **Integration**: Combining different systems/components
- **Inference**: Using trained model for prediction

### **J**
- **Joint**: Robot articulation point
- **Jacobian**: Mathematical relationship between joint and end-effector motion
- **Jerk**: Rate of change of acceleration

### **K**
- **Kinematics**: Study of motion without forces
- **Kinetic Chain**: Series of connected joints
- **Kalman Filter**: State estimation algorithm

### **L**
- **Language Model**: AI system understanding natural language
- **Locomotion**: Robot movement capability
- **LiDAR**: Light Detection and Ranging sensor

### **M**
- **Manipulation**: Robot object interaction
- **Middleware**: Software connecting different systems
- **ML (Machine Learning)**: AI system learning from data

### **N**
- **Neural Network**: AI system inspired by brain structure
- **Navigation**: Robot path planning and following
- **NVIDIA**: Technology company specializing in GPUs

### **O**
- **Open Source**: Freely available software
- **Omniverse**: NVIDIA's 3D simulation platform
- **Optimization**: Improving system performance

### **P**
- **Perception**: Robot environment sensing
- **Physics**: Simulation of real-world forces
- **Pipeline**: Series of processing steps

### **Q**
- **QoS (Quality of Service)**: Communication guarantees in ROS
- **Qdrant**: Vector database for embeddings
- **Quaternion**: Mathematical representation of rotation

### **R**
- **ROS (Robot Operating System)**: Robot middleware framework
- **Reinforcement Learning**: Learning through reward/penalty
- **Real-time**: System responding within time constraints

### **S**
- **Simulation**: Virtual robot environment
- **Sensor**: Device detecting environmental properties
- **SLAM**: Simultaneous Localization and Mapping

### **T**
- **Tensor**: Multi-dimensional array (mathematical)
- **Transformer**: Neural network architecture
- **Training**: Teaching AI system with data

### **U**
- **URDF**: Unified Robot Description Format
- **USD**: Universal Scene Description
- **Unity**: Game engine used for robotics

### **V**
- **VLA (Vision-Language-Action)**: Multimodal AI system
- **Vision**: Robot visual perception
- **Virtual Reality**: Immersive computer-generated environment

### **W**
- **Workspace**: Robot's operational area
- **Wheelchair**: Mobility aid (contrast with bipedal)
- **Whole-body Control**: Controlling entire robot simultaneously

### **X**
- **Xacro**: XML macro language for URDF
- **X,Y,Z**: Cartesian coordinate system axes

### **Y**
- **Yaw**: Rotation around vertical axis
- **YAML**: Human-readable data serialization format
- **Year**: Time period for development

### **Z**
- **ZMP (Zero Moment Point)**: Balance stability measure
- **Z-axis**: Vertical direction in 3D space
- **Zero-shot Learning**: Learning without examples
