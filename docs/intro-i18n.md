# Internationalization Support

This documentation site supports multiple languages including English and Urdu.

## Supported Languages

- **English** (LTR - Left to Right)
- **Urdu** (RTL - Right to Left)

## How to Switch Languages

You can switch between English and Urdu using the "اردو میں پڑھیں (Read in Urdu)" button that appears at the top of documentation pages. This button is only available to authenticated users.

## Authentication Requirement

To access Urdu translations, you must be logged in. The translation feature is gated to ensure only authorized users can access the localized content.

## Right-to-Left (RTL) Support

The site automatically adjusts layout and text direction when viewing Urdu content to provide the best reading experience.
