# Physical AI & Humanoid Robotics Platform

## Complete Course Documentation

Welcome to the comprehensive documentation for the **Physical AI & Humanoid Robotics Platform**. This course provides a complete learning pathway from fundamental robotics concepts to advanced AI-powered humanoid systems using NVIDIA's Isaac platform.

## üéØ Course Overview

This intensive course teaches students to build complete AI-powered humanoid robots using:
- **ROS 2 (Humble Hawksbill)**: Robot Operating System for communication
- **NVIDIA Isaac Sim**: GPU-accelerated simulation platform
- **Isaac ROS**: GPU-accelerated perception and navigation
- **Vision-Language-Action (VLA) Systems**: Multimodal AI for embodied intelligence
- **Humanoid Control**: Bipedal locomotion and manipulation

## üìö Complete Module Structure

### [Module 1: ROS 2 Fundamentals](./module-1-ros2/intro.md)
- **Week 1-2**: ROS 2 architecture, nodes, topics, services, actions
- **Week 3**: Python-ROS integration with rclpy
- **Week 4**: URDF for humanoid robot modeling
- **Week 5**: Middleware for real-time control

### [Module 2: Simulation Environments](./module-2-simulation/intro.md)
- **Week 6**: Physics simulation with Gazebo and PhysX
- **Week 7**: Unity for high-fidelity visualization
- **Week 8**: Sensor simulation and integration
- **Week 9**: Digital twin concepts and implementation

### [Module 3: NVIDIA Isaac Platform](./module-3-nvidia-isaac/intro.md)
- **Week 10**: Isaac Sim fundamentals and USD scenes
- **Week 11**: Isaac ROS integration and GPU acceleration
- **Week 12**: Isaac Gym for reinforcement learning
- **Week 13**: Synthetic data generation and domain randomization

### [Module 4: Vision-Language-Action Systems](./module-4-vla/intro.md)
- **Week 14**: VLA models and architectures
- **Week 15**: Streaming AI for real-time control
- **Week 16**: Humanoid-specific VLA implementations
- **Week 17**: Complete system integration

### [Capstone Project: Complete Humanoid System](./capstone-project/intro.md)
- **Week 18**: Integration of all modules
- **Week 19**: Performance optimization and validation
- **Week 20**: Deployment and testing

## ü§ñ Technical Stack

### Core Technologies
- **ROS 2 Humble Hawksbill**: Robot middleware and communication
- **NVIDIA Isaac Sim**: GPU-accelerated physics simulation
- **Isaac ROS**: GPU-accelerated perception and navigation
- **FastAPI**: Modern Python web framework for backend
- **Docusaurus**: Documentation and frontend framework
- **Qdrant**: Vector database for RAG systems
- **Gemini API**: Multi-modal AI for robotics

### Hardware Requirements
- **Minimum**: NVIDIA RTX 4070 (12GB VRAM), 32GB RAM, Ubuntu 22.04
- **Recommended**: NVIDIA RTX 4090 (24GB VRAM), 64GB+ RAM, 1TB+ SSD
- **Optimal**: NVIDIA A6000 or H100 for maximum performance
