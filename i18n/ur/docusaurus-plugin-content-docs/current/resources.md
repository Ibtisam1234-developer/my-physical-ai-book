## Glossary of Terms

### **A**
- **AI (Artificial Intelligence)**: Computer systems jo tasks perform करते हैं जिन्हें human-like intelligence चाहिए
- **API (Application Programming Interface)**: Software communication के लिए interface
- **Autonomous**: Self-governing, बिना human intervention के operate करने वाला

### **B**
- **Balancing Control**: Motion के दौरान robot stability बनाए रखना
- **Bipedal**: Locomotion के लिए दो legs वाला
- **Bridge**: Simulation और real systems के बीच connection

### **C**
- **CUDA**: NVIDIA का parallel computing platform
- **Computer Vision**: Image understanding पर focus करने वाला AI field
- **Cloud Robotics**: Cloud computing resources से connect robots

### **D**
- **Deep Learning**: Neural networks के साथ machine learning
- **Domain Randomization**: Sim-to-real transfer की technique
- **Dynamics**: Motion और forces का physics

### **E**
- **Embodied AI**: Physical presence वाले AI systems
- **End-Effector**: Robot arm के end पर tool या hand
- **Environment**: Robot के operational surroundings

### **F**
- **Foundation Model**: Large pre-trained AI model
- **Footstep Planning**: Robot अपने feet कहाँ रखे यह plan करना
- **Fusion**: Multiple sensor modalities को combine करना

### **G**
- **GPU (Graphics Processing Unit)**: Parallel processing hardware
- **Ground Truth**: Training के लिए correct reference data
- **Gym**: Reinforcement learning environment

### **H**
- **Humanoid**: Human-shaped robot
- **Human-Robot Interaction**: Human-robot communication का study
- **Hardware**: Physical robot components

### **I**
- **Isaac**: NVIDIA का robotics platform
- **Integration**: Different systems/components को combine करना
- **Inference**: Prediction के लिए trained model use करना

### **J**
- **Joint**: Robot articulation point
- **Jacobian**: Joint और end-effector motion के बीच mathematical relationship
- **Jerk**: Acceleration की change की rate

### **K**
- **Kinematics**: Forces के बिना motion का study
- **Kinetic Chain**: Connected joints की series
- **Kalman Filter**: State estimation algorithm

### **L**
- **Language Model**: Natural language समझने वाला AI system
- **Locomotion**: Robot movement capability
- **LiDAR**: Light Detection and Ranging sensor

### **M**
- **Manipulation**: Robot object interaction
- **Middleware**: Different systems को connect करने वाला software
- **ML (Machine Learning)**: Data से सीखने वाला AI system

### **N**
- **Neural Network**: Brain structure से inspired AI system
- **Navigation**: Robot path planning और following
- **NVIDIA**: GPUs में specialize करने वाली technology company

### **O**
- **Open Source**: Freely available software
- **Omniverse**: NVIDIA का 3D simulation platform
- **Optimization**: System performance improve करना

### **P**
- **Perception**: Robot environment sensing
- **Physics**: Real-world forces का simulation
- **Pipeline**: Processing steps की series

### **Q**
- **QoS (Quality of Service)**: ROS में communication guarantees
- **Qdrant**: Embeddings के लिए vector database
- **Quaternion**: Rotation का mathematical representation

### **R**
- **ROS (Robot Operating System)**: Robot middleware framework
- **Reinforcement Learning**: Reward/penalty से सीखना
- **Real-time**: Time constraints के भीतर respond करने वाला system

### **S**
- **Simulation**: Virtual robot environment
- **Sensor**: Environmental properties detect करने वाला device
- **SLAM**: Simultaneous Localization and Mapping

### **T**
- **Tensor**: Multi-dimensional array (mathematical)
- **Transformer**: Neural network architecture
- **Training**: Data के साथ AI system teach करना

### **U**
- **URDF**: Unified Robot Description Format
- **USD**: Universal Scene Description
- **Unity**: Robotics के लिए use किया जाने वाला game engine

### **V**
- **VLA (Vision-Language-Action)**: Multimodal AI system
- **Vision**: Robot visual perception
- **Virtual Reality**: Immersive computer-generated environment

### **W**
- **Workspace**: Robot का operational area
- **Wheelchair**: Mobility aid (bipedal के विपरीत)
- **Whole-body Control**: Entire robot को simultaneously control करना

### **X**
- **Xacro**: URDF के लिए XML macro language
- **X,Y,Z**: Cartesian coordinate system axes

### **Y**
- **Yaw**: Vertical axis के around rotation
- **YAML**: Human-readable data serialization format
- **Year**: Development का time period

### **Z**
- **ZMP (Zero Moment Point)**: Balance stability measure
- **Z-axis**: 3D space में vertical direction
- **Zero-shot Learning**: Examples के बिना सीखना
