# VLA نفاذ کے نمونے

Vision-Language-Action (VLA) ماڈلز کو نافذ کرنے کے مختلف نمونے ہیں۔

## عملی نفاذ کے طریقے

### 1. اسٹیجائزڈ تربیت

- **وژن ماڈل**: تصاویر کو تیار کرنے کے لیے
- **زبانی ماڈل**: ٹیکسٹ کو سمجھنے کے لیے
- **ایکشن ماڈل**: حرکت کو تیار کرنے کے لیے

### 2. یک جامع ماڈل

- تمام ماڈلز کو ایک نیٹ ورک میں جوڑ دیا جاتا ہے
- کمپیوٹیشنل وسائل کی کم خوراک
- بہتر تعاون کے لیے

### 3. ماڈلر ماڈلز

- ہر ماڈل کو الگ الگ تربیت دی جاتی ہے
- پھر انہیں ایک سسٹم میں ضم کر دیا جاتا ہے
- زیادہ لچک دار، لیکن پیچیدہ ہے

## عملی اطلاق

 humanoid robot کے لیے VLA نفاذ کے عملی اطلاق کا جائزہ لیں۔
